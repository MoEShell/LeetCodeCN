# 括号生成
### 思考
这一题明显有三个限制：
 1. '(' 个数为 n
 2. ')' 个数与 '(' 相等
 3. 总长度为 2n

对应在生成过程中：
 1. '(' 个数不超过 n
 2. ')' 个数不超过 '('
 3. 总长度不超过 2n

可以构造一个递归过程，当有空位时，如果 '(' 数不多于 n 空位可以填 '('。如果')' 数不多于 '(' 空位可以填 ')'

### 算法
 0. 构造的字符串 str，'(' 数 open，')' 数 close，个数 n
 1. 如果 str 长度等于 2n，构造完成
 2. 如果 str 长度小于 2n
    1. 如果 open 小于 n，str + '('，重复 2、3
    2. 如果 close 小于 open str + ')'，重复2、3
